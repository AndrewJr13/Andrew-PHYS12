I tested my phone, a piece of metal, and a drill. ![](./table.png) And here is the code I used for the sensor. <pre><code class="language-arduino"> int analog_pin = A3; int tx_pin = 4; int read_high; int read_low; int diff; void setup() { pinMode(tx_pin, OUTPUT); Serial.begin(9600); } void loop() { digitalWrite(tx_pin,HIGH); read_high = analogRead(analog_pin); delayMicroseconds(100); digitalWrite(tx_pin,LOW); read_low = analogRead(analog_pin); diff = read_high - read_low; Serial.println(diff); } </code></pre> So that was the first sensor I built.
